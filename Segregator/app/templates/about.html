ABOUT

The project "Automatic Waste Segregator" has been developed by the students of Dayananda Sagar College of Engineering , Bangalore namely , Boudhayan Dev, Aditya Agarwal, Chinmay Hebbal , Aishwarya H.S. The device is capable of detecting , identifying and segregating waste items without any human intervention.

The prime motivation behind this project has been the problem that affects us all i.e Waste Management. The million tons of waste that are generated everyday, do not get properly segreagted and as a result large portion of the non biodegradable waste items are mixed with biodegradable ones and are dumped in landmines. This ultimately piles up and their effect on the environment esp. soil and groundwater resources becomes devastating. Thus, we have come up with the idea that if we can segregate the waste items at the primary level itself, then the percentage of mixup can be reduced by a great margin. This in turn will also help in increasing the amount of recyclable items from the waste heap.

The major components of our project involve the following parts :-
  • Image processing
  • Robotics
  • Robotics
  • Machine Learning
  • Web server

Image Processing


Raspberry Pi 3 is the CPU for the entire system. For Image processing, we have used PiCamera v.2 and OpenCV module to perform the background subtraction. Background subtraction was used to detect any object's presence while the device travels in its path.




Background Subtraction also pinpoints the object's location in the image frame. The location is used by the robotic arm to pick up the object.

Robotics


The robotic arm is composed of 4 joints :-
  • Shoulder joint.
  • Elbow joint.
  • Wrist joint.
  • Finger joint.

    • The shoulder joint positions the arm over the quadrant where the object lies. The elbow joint pickts up the object from the ground and the finger+wrist assembly helps to contract and expand the finger joints.


    The device is supported by 2 wheels at the base which is powered by DC motors. The wheels help the device to navigate in a given area.

    Machine Learning

    The identification of the waste category is performed with the help of Deep Neural Networks. We have trained a dataset of 50,000 images belonging to biodegradable and non-biodegradable categories. The Nerual network was trained using Transfer Learning.

    Web server


    The final component is the Web server which serves as the control center for the entire system. Navigate to Dashboard, to view the test results obtained during the trial phase.This webserver is built using Flask framework which displays real-time operation of the system.
